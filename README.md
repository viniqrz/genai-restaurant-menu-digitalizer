# genai-restaurant-menu-digitalizer

Minimal toolkit for converting photographed or scanned restaurant menus into structured JSON using OCR + generative prompts.

See `notebook.ipynb` for an interactive demo.

Quick start:

1. Clone the repo and open `notebook.ipynb` in Jupyter or VS Code.

2. (Optional) Create a venv and install dependencies you add (e.g., `pip install -r requirements.txt`).
